## 一、什么是代币化股票

代币化股票（Tokenized Stocks）是把传统美股 / 港股**通过合规托管方 1:1 抵押**后发行的链上代币：

- 每枚代币 = 1 股真实股票（或 0.x 股，按发行规则）
- 价格与真实股票实时联动
- 24/7 全天可交易（不受美股盘前盘后限制）
- 用稳定币（USDC / USDT）直接计价
- 链上结算 + 链上托管，去中心化部分操作

OKX 与多个合规发行方（如 Backed Finance、Dinari）合作上线代币化美股 / 港股。

## 五、X 平台高频踩坑警告

- **代币化 ≠ 真实持股**：你持有的是代币，不是真实股票。**没有投票权、部分股息有差异、合规框架可能变化**
- **抵押品风险**：代币化股票依赖发行方的合规托管，**发行方破产 / 监管收紧可能影响代币赎回**
- **流动性深度浅**：代币化股票流动性远不如真实股票，**大额单可能滑点严重**
- **支持股票清单有限**：目前仅覆盖部分主流美股 / 港股，**冷门股 / 新股可能没有代币化版本**
- **24/7 交易陷阱**：盘后流动性差，**剧烈波动时滑点放大**，新手别在凌晨开大单
- **税务申报**：代币化股票收益**可能被认定为加密资产**，**报税方式与真实美股不同**
- **国内 IP 不可用**：OKX 国内版无代币化股票，**必须使用国际版 + 海外网络环境**
- **合规地区动态变化**：每年都有新的国家被纳入或退出代币化股票合规名单，**注册前查 OKX 公告**
