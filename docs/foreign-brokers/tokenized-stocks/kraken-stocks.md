## 六、X 平台高频踩坑警告

- **大陆 IP 不可用**：Kraken xStocks 严格限合规地区，**大陆 IP 注册成功后 KYC 阶段被拒**
- **客服全英文**：复杂问题需邮件，平均响应 24~48h
- **资金来源审查严格**：大额入金需提供完整资金链路，**P2P 来源资金可能被冻结**
- **合规地区动态变化**：合规名单按各国监管动态调整，**注册前查 Kraken 公告**
- **xStocks 服务条款额外问卷**：与普通 KYC 分开，**漏填即不开通**
- **流动性深度浅**：相比 Binance，Kraken xStocks 流动性中等，**大额单滑点高**
- **税务申报**：Kraken 自动生成欧盟 / 英国税务表格，**美国税务居民不可用**
- **24/7 不等于全天流畅**：盘后 / 节假日深度大幅下降
