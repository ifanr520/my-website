## 四、X 平台高频踩坑警告

- **代币化 ≠ 真实持股**：与其它代币化股票同理，**没有投票权 + 部分股息差异**
- **支持股票数量有限**：Bybit 代币化股票数量比 OKX / Binance 略少
- **代币化股票退市风险**：发行方调整可能影响代币赎回
- **盘后流动性差**：节假日 / 凌晨深度大幅下降
- **大陆 IP 不可用**：必须海外网络环境
- **合规地区动态变化**：每年都有新国家被纳入 / 退出名单
- **税务申报**：与真实股票分类不同，**报税方式注意**
