## 五、X 平台高频踩坑警告

- **代币化 ≠ 真实持股**：Binance 代币化股票同样是合规托管，**没有投票权 + 部分股息差异 + 合规风险**
- **支持股票数量有限**：仅覆盖头部美股，**中小盘股没有代币化版本**
- **盘后流动性陷阱**：盘后 / 节假日流动性虽然不间断，**但深度大幅下降**，大额单滑点严重
- **代币化股票退市风险**：发行方可能因合规调整下架某些股票，**链上代币需赎回为现金**
- **税务申报混淆**：代币化股票收益**报税分类与真实股票不同**，部分税务局认定为加密资产收益
- **大陆 IP 不可用**：Binance 国内版无代币化股票，**必须国际版 + 海外网络**
- **杠杆陷阱**：部分代币化股票可加杠杆，**新手不要尝试**
- **客服钓鱼**：Binance 官方**永远不在 Telegram / X 私信**
