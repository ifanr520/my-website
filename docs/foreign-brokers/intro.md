# 国外券商 · 栏目说明

本栏目用于沉淀境外券商的开户与长期使用经验，覆盖以下方向：

- 盈透 IBKR、嘉信 Schwab、Tiger、长桥、富途、Webull 等主流券商的开户路径
- 银证转账、电汇、ACH、Wise、加密卡入金等不同资金通道
- W-8BEN、CRS、税务表格的填写与年度回顾
- 账户安全（双因素认证、白名单、冷热分离）与长期仓位记录方式

> 本栏目正在持续整理中，最新攻略发布后会自动出现在左侧目录与顶部导航中。
