## 七、X 平台高频踩坑警告

- **填表中断需要重填**：申请表很长，**一次性填完**避免中断；浏览器 cookie 关闭也会导致需要重填
- **大陆 IP 注册受限**：部分大陆 IP 注册成功后，**KYC 阶段被拒**，建议海外网络环境
- **W-8BEN 误填美国税务居民**：与富途同理，**误填后需重新签**
- **Trading Permissions 选错**：申请时漏选某市场，后续要"提交补充申请 + 客服审核"才能开通
- **TWS 学习曲线陡**：TWS 桌面终端功能极强但 UI 复杂，**新手用 IBKR Mobile App 即可**
- **IBKR Lite vs Pro 切换**：Lite 切换到 Pro 即可，但 Pro 切回 Lite 有 30 天冷却期
- **客服全英文**：IBKR 客服主要英文，**亚太地区有部分中文客服**但响应慢
- **Pattern Day Trader 规则**：< 25,000 USD 触发 PDT，**强制冻结 90 天**
- **数据费按月订阅**：行情数据按市场订阅，**不需要的市场不订阅省钱**
- **大额入金审查**：> 100,000 USD 入金需提供资金来源证明
