# 漂哥出海之路

这里是「资源全球运营与布局」的出海攻略知识库。所有内容按六个固定栏目沉淀，方便从账户、资产、支付、增长、AI 工具与视觉内容多个方向持续查找。

## 当前目录

- **国外银行**：海外数字银行开户与加密银行卡的入金路径、费率结构、风控触发点。
- **国外券商**：境外券商开户、交易费率、税务表格、账户安全与长期仓位记录。
- **加密 Web3**：交易所、钱包、稳定币、链上工具与基础安全习惯。
- **跨境出海**：主体设立、跨境收付、内容平台、增长渠道与运营工具栈。
- **AI 时代**：AI 写作、调研、自动化、编程辅助与个人效率系统。
- **AI 美女相册**：AI 视觉创作与提示词工程归档，参数、风格与迭代记录。

## 内容更新方式

把 Markdown 文件放到对应文件夹即可，左侧目录与顶部导航会**自动同步**：

```text
docs/
  foreign-banks/
    digital-banks/        # 数字银行开户
    crypto-cards/         # 加密银行卡
  foreign-brokers/
  crypto-web3/
    platform-guides/      # 交易所与平台
  cross-border/
    playbooks/            # 落地攻略
  ai-era/
  ai-gallery/
    creative-guides/      # 创作 Prompt
```

每篇文章建议保留清晰的 front matter：

```md
---
sidebar_label: 文章短标题
sidebar_position: 2
description: 文章一句话摘要，会出现在卡片列表与 SEO meta 中。
---

# 文章完整标题
```
