# 漂哥出海之路

这里是“资源全球运营与布局”的出海攻略知识库。网站内容会按固定栏目沉淀，方便读者从账户、资产、支付、增长、AI 工具和视觉内容几个方向持续查找。

## 当前目录

- 国外银行：海外银行开户、资金出入、费用、账户维护和风控经验。
- 国外券商：券商开户、入金提现、税务表格、账户安全和长期配置笔记。
- 加密 Web3：交易所、钱包、稳定币、链上工具和 Web3 基础设施。
- 跨境出海：公司、支付、内容平台、海外增长和运营工具。
- AI 时代：AI 写作、调研、自动化、编程辅助和效率系统。
- AI 美女相册：AI 图片生成、提示词、视觉栏目和作品归档。

## 新增文章方式

把 Markdown 文件放到对应文件夹即可，左侧目录会自动归档：

```text
docs/
  foreign-banks/
  foreign-brokers/
  crypto-web3/
  cross-border/
  ai-era/
  ai-gallery/
```

每篇文章建议保留清晰的 front matter：

```md
---
sidebar_label: 文章短标题
sidebar_position: 2
---

# 文章完整标题
```
