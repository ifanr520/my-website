## X 平台高频踩坑

- **FUP 公平使用条款**：超过 50GB / 月会限速到 1Mbps（Pro 套餐 100GB），<strong>视频会议党建议直接选 Pro</strong>
- **自动续订陷阱**：默认开启月度续订，<strong>不需要时务必去 App 关闭</strong>，不然每月自动扣 $30
- **新国家激活延迟**：到新国家后<strong>需 5-30 分钟自动切换运营商</strong>，期间可能完全没信号
- **iOS eSIM 槽位**：长期使用建议<strong>独占一个 eSIM 槽位</strong>，避免和别的 eSIM 切换
- **覆盖国家有限**：非洲 / 中东 / 拉美部分国家不在 90 国列表内，<strong>下单前先看覆盖列表</strong>
- **退款政策**：开通后 14 天内未激活可全额退款；激活后无法退款
- **网络速度因国而异**：东南亚 50-100Mbps、欧洲 100Mbps+、北美 50-80Mbps（部分网络拥堵地区慢）
