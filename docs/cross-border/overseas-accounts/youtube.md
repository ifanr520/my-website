# 中国大陆用户注册 YouTube 频道全攻略

YouTube 是<strong>跨境内容创作者赚美金最稳的平台</strong>，AdSense 收入按 CPM 美金结算，远高于国内。但 YouTube 等于 Google 全家——所有 Google 风控规则都适用，注册 + 开通 + 上传 + 收款<strong>四道关卡都有坑</strong>。这篇按 X、Reddit r/NewTubers 与跨境创作者社群反馈，把整套路径压到一篇。

## 一句话结论

| 用户画像 | 推荐路径 |
|---------|---------|
| 想做内容赚 AdSense | 海外 Gmail + 美国地址 + 海外银行账户收款 |
| Vlog / 教学 | 同上 + 头 100 视频不商业化 |
| 短视频 Shorts | 同上 + 频道开通后立刻发 5+ 视频 |
| 单纯看 / 上传给朋友 | 任何 Gmail 直接登录 youtube.com |

> YouTube 开通"<strong>看视频</strong>"几乎无门槛；难的是<strong>开通频道、激活获利、收款到中国大陆</strong>三步。

## 一、注册前提

### 你需要先有这些（详见相关攻略）

1. **美国 Gmail 账号** — 详见 [美国 Gmail 注册攻略](/docs/cross-border/overseas-accounts/gmail-us)
2. **海外身份信息**（用于 AdSense W-8BEN / 美国地址）
3. **海外银行账户**（收 AdSense 美金，详见 [iFast](/docs/foreign-banks/digital-banks/ifast-global-bank) / [Wise](/docs/foreign-banks/fintech-platforms/wise)）
4. **海外 IP 环境**（注册 + 开通 + 上传都要稳定海外节点）

> ⚠️ 中国大陆 IP 看 YouTube 没问题（YouTube 不限大陆 IP 看视频，只是被防火墙挡）；但<strong>开通频道 + 上传视频 + 激活获利</strong>必须挂海外 IP。

---

## 二、开通频道

YouTube 频道有两种：

### A. 个人频道（关联个人 Google 账号）

最简单的路径：

1. 海外 Gmail 登录 [youtube.com](https://www.youtube.com/)
2. 右上角点头像 → Create a Channel
3. 频道名（中英文均可）+ 头像
4. 完成开通，立即可上传

**适合**：业余分享 / 学习记录 / 不想商业化

### B. 品牌账号频道（推荐）

把频道独立于 Gmail，未来可以拉团队成员一起管理：

1. Gmail 登录 YouTube
2. 头像 → Settings → Add or manage your channel(s) → Create new channel
3. 选择 "Use a brand account name"
4. 输入品牌名 + 描述
5. 频道成员管理：可邀请其他 Gmail 协作

**适合**：长期内容创作 / 团队运营 / 商业化

## 三、频道初始化（注册当天必做）

频道刚开通时，<strong>立即完成以下设置</strong>，提升算法初始权重：

1. **头像 + 频道横幅**（建议设计简洁，不要纯文字）
2. **频道描述**（200 字英文，介绍内容方向）
3. **频道关键词**（YouTube Studio → Settings → Channel → Basic info → Keywords）
4. **链接**（个人网站 / IG / X）
5. **国家选 United States**（决定 CPM 单价 + AdSense 计费货币）
6. **隐私设置**：关闭"Notify subscribers when uploading"（让你能悄悄发视频测试）

---

## 四、上传第一条视频（关键）

YouTube 算法对<strong>头 5 条视频</strong>给的权重决定频道未来：

### 视频质量基本要求

- **分辨率 ≥ 1080p**（4K 更好）
- **时长 ≥ 8 分钟**（中长视频比短视频获利门槛低）
- **标题 + 描述全英文**（中文也可，但英文 CPM 高 5-10x）
- **缩略图自定义**（默认截图必死）
- **添加章节标记**（Chapters，提升 watch time）

### 上传步骤

1. YouTube Studio → Create → Upload
2. 选文件
3. 标题（含 1-2 个关键词，60 字符内）
4. 描述（≥ 100 字，含 5+ hashtag）
5. 缩略图
6. 标签（5-10 个）
7. 分类
8. 字幕（自动 / 手动）
9. **Audience**：选 "No, it's not made for kids"（除非你做儿童内容）
10. 隐私：先 Unlisted 测试 → 确认无问题再 Public

> 头 24 小时<strong>不要立刻删视频</strong>，YouTube 会标记"频道质量不稳定"。

## 五、激活获利（YouTube Partner Program）

### 加入条件（2025 标准）

- **订阅 ≥ 1,000**
- **公开视频累计观看时长 ≥ 4,000 小时**（过去 12 个月）
- 或 **Shorts 累计观看 ≥ 10,000,000**（过去 90 天）
- 频道 0 警告
- 链接 AdSense 美元账户
- 启用 2FA

### 申请流程

1. YouTube Studio → Earn → 满足条件后立即可申
2. **AdSense 关联**：跳转 google.com/adsense 创建 AdSense 账号
3. AdSense 国家选 **United States**（CPM 高最高）
4. 填美国地址 + 美国电话
5. 提交后<strong>等 1-30 天审核</strong>

### W-8BEN 税表（必填）

非美国税务居民必须提交 W-8BEN：

- AdSense → Payments → Tax info → Add tax info
- 选 Individual Non-US
- 填中国 / 海外居住信息
- 美国预扣税：默认 30%，<strong>填中国税号可享受 10% 协定税率</strong>（仅 YouTube AdSense 适用）

## 六、收款到中国大陆（最关键）

AdSense 收款需要海外银行账户。中国大陆账户接收 AdSense 美金<strong>近年越来越严</strong>：

### 路径 A：Wise USD 账户（推荐）

1. Wise 多币种账户开通 USD 子账户（详见 [Wise 攻略](/docs/foreign-banks/fintech-platforms/wise)）
2. AdSense 收款方式 → Wire transfer / ACH
3. 填 Wise 美国 ACH 账号 / Routing number
4. 月底 AdSense 自动打款 USD 到 Wise
5. Wise 可换汇为 RMB 提现到国内银行

### 路径 B：iFast Global Bank（推荐）

1. iFast 英镑账户接收 ACH（USD 转 GBP）
2. iFast 月底持有 USD → 换汇 GBP / SGD 后提现
3. 详见 [iFast 攻略](/docs/foreign-banks/digital-banks/ifast-global-bank)

### 路径 C：BiyaPay（已有 USDT 用户）

- BiyaPay 平台支持收 USD AdSense（部分用户可用）
- USDT 互转灵活

### 路径 D：海外公司账户

- 美国 LLC / 香港有限公司 / 新加坡公司账户接收
- 适合<strong>大额收入用户</strong>（月 ≥ $10,000）

## 七、X 平台与 Reddit 高频踩坑

- **大陆 IP 上传视频被风控**：YouTube 风控会标记上传 IP，<strong>大陆 IP 上传 5+ 视频后频道质量下降</strong>
- **Gmail 是新号 + YouTube 立即开频道**：Google 风控触发，频道被立刻 disabled
- **频道国家选 China**：CPM 低 80%，且<strong>无法激活获利</strong>，必选 US
- **W-8BEN 没填 = 30% 预扣税**：填了 W-8BEN 享 10% 协定税率，省 20%
- **Shorts 与长视频混发**：频道初期建议<strong>专注一种格式</strong>，混发会让算法分不清你的目标受众
- **音乐版权警告 = 整频道降权**：哪怕一次 Copyright Strike 也会让频道初始权重清零，<strong>务必用 YouTube Audio Library 免费音乐</strong>
- **大陆梯子上传超大文件失败**：4K 视频 50GB+ 文件上传，<strong>梯子带宽是瓶颈</strong>，建议租 VPS 或用海外 eSIM 流量
- **AdSense 拒批**：常见原因 — 内容重复 / 视频太少 / 频道时间太短 / 真实身份不一致
- **频道两次警告 = 永久 disabled**：YouTube 不像 IG 可以申诉，第三次警告 = 永久封
- **新频道立即开 Shorts 不发长视频**：算法不知道你是什么频道，CPM 低
- **接码号绑定 Gmail = AdSense 风控**：接码号注册的 Gmail 在 AdSense 阶段被识别 = 拒批
- **直播弹幕 / 评论中文广告**：被举报后封号
- **标题 / 描述堆关键词**：触发"misleading metadata"扣分

## 八、Shorts vs 长视频策略

| 维度 | YouTube Shorts | 长视频 |
|------|----------------|--------|
| 内容形态 | < 60 秒竖屏 | > 8 分钟 |
| 流量门槛 | 低（新号也有流量）| 高（要靠 SEO + 推荐）|
| 商业化门槛 | 1 千万播放 / 90 天 | 4000 小时 / 12 个月 |
| CPM | $0.05-$0.20 | $1-$10 |
| 适合 | 引流 + 涨粉 | 赚 AdSense |

**结论**：Shorts 涨粉、长视频赚钱，<strong>两者结合是最稳的</strong>。

## 总结

中国大陆做 YouTube 赚 AdSense 美金的<strong>三大门槛</strong>：

1. **注册不被风控** — 海外 Gmail + 海外 IP + 全新设备
2. **激活获利** — 1k 订阅 / 4k 小时（或 Shorts 1000 万播放），最快 3 个月
3. **收款落地** — Wise / iFast / 海外公司账户，<strong>不能直接收到国内卡</strong>

按本文走一遍，<strong>3-6 个月稳定开始月入 $200+</strong>，6-12 个月有机会破 $1k/月。
