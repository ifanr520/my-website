# 中国大陆用户注册 Facebook 全攻略

Facebook 是<strong>跨境电商投广告、出海找海外人脉、加海外社群的根账号</strong>。但 Meta 风控比 Instagram 还激进——<strong>很多新号注册当天就被"提交身份证件"或直接 disabled</strong>。这篇按 X、Reddit r/facebook 与跨境电商社区反馈，把成功率最高的路径压到一篇。

## 一句话结论

| 用户画像 | 推荐路径 |
|---------|---------|
| 投广告 / 跨境电商 | 海外身份户 + 海外公司主体 BM |
| 个人社交 / 找海外朋友 | 海外 eSIM + 真名头像 + 加 5+ 老朋友 |
| Marketplace 二手交易 | 海外身份 + 一年以上老号 |
| 单纯做备份 | 注册后只看不发，每月登一次 |

> Facebook 是<strong>所有海外社交账号里风控最严的</strong>。新号 30 分钟内被风控的比例约 50%。

## 一、注册前准备

### 1. 网络

| 节点 | 通过率 |
|------|------|
| 美国 / 加拿大住宅 IP（海外 eSIM 流量）| **最高** |
| 日本 / 澳大利亚住宅 IP | 高 |
| 欧盟 IP | 中（GDPR 多一道流程）|
| 数据中心 IP / 公共 VPN | 极低 |
| **香港 / 新加坡** | 极低 |
| 大陆 IP | 0 |

### 2. 设备

- **iPhone Safari 注册**通过率最高
- **新设备 / 没装过 FB App 的手机**最优
- 安卓老设备清广告 ID + 重置 Google Play
- ❌ iCloud 多账号设备 — Meta 容易关联识别

### 3. 海外手机号

- ✅ giffgaff / Vodafone DE / 3HK 自有 eSIM 号（最稳）
- ⚠️ 接码平台 sms-activate（通过率 ~30%）
- ❌ +86 国内号

### 4. 邮箱

- ✅ <strong>已用 ≥ 6 个月的老 Gmail（美区）</strong>是首选
- ❌ 全新 Gmail：注册 IG / FB 当天，新 Gmail 也容易被怀疑
- ❌ 国内邮箱直接拒

### 5. 真实身份要素（关键）

Facebook 注册<strong>"看起来像真人"</strong>极重要：

- **真名 + 真实头像**（拼音英文名 + 一张真人照）
- **真实出生日期**（≥ 25 岁通过率最高，新号最低 18 岁）
- **简介填基本工作 / 教育**（不要纯空白）
- 准备 5-10 张 <strong>本地相册照片</strong>（人物、风景、生活）

## 二、注册步骤

### 路径 A：iOS App（推荐）

1. iPhone 美区 Apple ID → App Store 下载 Facebook
2. **关 WiFi 切到海外 eSIM 数据流量**
3. 打开 App → Sign Up
4. 拼音英文名 + 真实生日 + 性别
5. 海外手机号 + SMS 验证
6. 老 Gmail 邮箱 + 邮箱链接验证
7. 设置密码 ≥ 12 位
8. **立即上传头像**（关键！）
9. **立即填工作 / 教育简介**
10. **添加 5+ 已有 FB 朋友**（最关键）

### 路径 B：网页注册

1. Safari 打开 [facebook.com](https://www.facebook.com/)
2. Sign Up → 填表
3. 通过率比 App 略低

### 步骤 11：好友冷启动（最最重要）

Facebook 算法极其重视"好友图"。新号<strong>注册当天 + 头 3 天内必须加 5-15 个真实好友</strong>，否则被判定为机器人号：

- ✅ 国内已经有 FB 的朋友 / 同学 / 同事
- ✅ 海外华人留学生群里互加
- ✅ 已加的某个 FB 群组里互相加好友
- ❌ 加陌生人 / 加"建议好友"列表 — 算法会怀疑你是营销号

> 没有 5+ 真实好友的新号，<strong>30 天内被封率 80%+</strong>。

## 三、头 30 天养号策略

### Week 1
- 加 5+ 真实好友
- 上传头像 + 简介
- 浏览首页 5-10 分钟 / 天
- 给好友帖子点 3-5 个赞
- ❌ 不加群、不发帖、不发广告

### Week 2
- 加 5+ 个 FB Group（兴趣 / 爱好 / 跨境运营群）
- 在群里看 + 偶尔点赞
- 发 1 条 Story（生活照）
- 给好友评论 1-2 条普通评论

### Week 3-4
- 发<strong>第一条 Post</strong>（图片 + 英文简单文案）
- 在 1-2 个 Group 里发普通评论
- 加更多 Group
- 关注 5-10 个 Public Page

30 天后号比较稳了。

## 四、X 平台与 Reddit 高频踩坑

- **注册当天就要求"提交身份证件"**：Meta 直接要求 ID Verification — 这是最常见情况，<strong>没法绕过，提交护照通过率约 60%</strong>
- **注册 30 分钟内 disabled**：网络环境 / 设备指纹被识别 = 风控立即触发
- **同设备多账号**：FB 一台设备最多 1 个号，多号同设备 = 全家被封
- **VPN 跳节点**：注册时 IP 美西、登录时 IP 日本 → 风控警告
- **批量加好友 / 加群组**：一天 ≥ 30 个加好友请求 = 立即风控
- **新号立即开 BM 投广告**：广告账号会被无限期审核，绑定身份证件
- **iCloud 备份恢复设备**：旧 iPhone 注册过 FB 被封，新 iPhone iCloud 恢复后注册新号 = 立即关联封号
- **拍照发帖泄漏 GPS**：FB 自动读 EXIF，照片 GPS 是中国 = 风控扣分
- **接码号注册成功，2 周后被风控**：接码号被多人复用，FB 关联识别
- **同一手机号绑过 IG 和 FB**：建议<strong>IG 用号 A、FB 用号 B</strong>，分开管理
- **被封号申诉成功率**：< 20%，<strong>建议直接换设备 + 换号重来</strong>
- **Marketplace 在大陆 IP 不显示**：必须挂海外 IP 才能看到 Marketplace 入口

## 五、Business Manager（投广告必备）

跨境电商投 Facebook / IG 广告需要 BM 账号：

### 个人 BM（不推荐）

- 用个人 FB 账号开 BM
- 风控严，<strong>新号开 BM 立即被锁</strong>
- 仅适合<strong>FB 老号 ≥ 1 年</strong>

### 海外公司 BM（推荐）

- 用海外公司主体（美国 LLC / 香港有限公司 / 新加坡公司）开 BM
- 提交营业执照 / 公司信息
- BM 通过后绑定海外银行卡 / 美国 EIN
- **稳定性远高于个人号**

### 托管 BM（最快）

- 找广告代理商代开 BM
- 月租 $200-$500
- 可立即投广告
- 适合<strong>不想自己折腾的卖家</strong>

## 六、Facebook + Instagram 关联

- IG 与 FB 同属 Meta，可以关联但<strong>不建议立即关联</strong>
- 注册成功后 30 天再考虑关联（避免一边封号波及另一边）
- 关联后可<strong>同发 Reels / Story / Post</strong>到两边

## 七、账号安全

1. Settings → Security 开 **2FA**（Authenticator）
2. 备用邮箱
3. **Trusted Contacts**（紧急联系人，找回账号用）
4. 关闭 Activity Sharing（不让别人看到你的活动）
5. 头像 / 封面图设私密查看

## 总结

中国大陆用户注册 FB 的关键是<strong>"做一个真人朋友圈"</strong>：

1. **海外 eSIM + 全新设备**硬件指纹
2. **拼音英文真名 + 真人头像 + 工作教育简介**账户外观
3. **注册当天加 5+ 真实好友**冷启动（最最重要）
4. **30 天养号 + 不发广告 + 不批量操作**行为指纹

按本文走一遍，<strong>30 天通过率约 60%，6 个月稳定使用率 80%+</strong>。
