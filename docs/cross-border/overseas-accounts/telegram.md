## 一、注册前提

### 1. 网络

Telegram 在中国大陆<strong>被防火墙完全屏蔽</strong>，必须挂海外 IP：

| 节点 | 通过率 |
|------|------|
| 美国 / 欧盟 / 日本 / 新加坡 | 高（任意挂稳定即可）|
| 大陆裸 IP | 0 |

> Telegram 对节点选择不像 Google / Meta 那么挑剔——能上就行，但<strong>挂稳定节点</strong>避免登录时频繁掉线。

### 2. 手机号选择（最关键）

Telegram 要求 SMS 验证，号码选择决定你账号长期使用体验：

| 号码类型 | 通过率 | 推荐度 |
|---------|------|------|
| **海外 eSIM 实体号**（giffgaff / 3HK / Clubsim）| 99% | ⭐⭐⭐⭐⭐ |
| 借朋友海外手机号 | 99% | ⭐⭐⭐⭐⭐ |
| **+86 国内手机号** | **70%**（首次能注册）| ⭐⭐（功能受限）|
| sms-activate 接码 | 80% | ⭐⭐⭐ |
| Google Voice | 95% | ⭐⭐⭐⭐ |
| **Fragment +888 anonymous 号** | 100% | ⭐⭐⭐⭐⭐（隐私党推荐）|

> ⚠️ **+86 注册后的限制**：无法加入 / 创建大群、无法发送私信给陌生人、无法订阅 Premium、容易被 Telegram 自动标记"垃圾号"。**强烈推荐自有海外 eSIM**。

### 3. 设备

任何 iPhone / 安卓 / Mac / Windows 都行，Telegram 设备风控最宽松。

## 二、注册步骤

### 路径 A：iOS App（最常用）

1. App Store 下载 Telegram（任何区 Apple ID 都行）
2. 挂海外 VPN
3. 打开 App → Start Messaging
4. 选择国家 + 输入手机号
5. 接收 SMS 6 位验证码 → 输入
6. 设置姓名（可中文 / 英文 / 拼音）
7. 设置头像
8. 完成注册，立即可用

### 路径 B：Telegram Desktop（PC / Mac）

1. 下载 [desktop.telegram.org](https://desktop.telegram.org/)
2. 流程同上

### 路径 C：Web（[web.telegram.org](https://web.telegram.org/)）

- 不需要装 App，浏览器直接用
- 但<strong>不支持 Premium / Wallet / 部分新功能</strong>

### 适合

## 五、X 平台与 Reddit 高频踩坑

- **+86 注册后想换号**：Settings → Account → Change Number，<strong>历史聊天记录 / 群组都会同步到新号</strong>，无缝迁移
- **接码平台号 7 天后失效**：你买的号被运营商回收 → Telegram 触发 SMS 重新验证 → 失败 → 账号被锁。**自有 eSIM > 接码平台**
- **多设备登录**：Telegram 默认支持手机 + 电脑同时登录；但<strong>同一账号过多设备会被风控</strong>，建议保持 ≤ 5 个设备
- **加群 / 加频道频率**：一天加 ≥ 200 个群 / 频道 = 触发风控，被限制操作 24h
- **批量私信**：发 ≥ 50 条私信给陌生人 = 立即风控，账号被标记 "spambot"
- **群组里发广告**：被群主或机器人举报 → 触发 spam ban，账号无法发私信给陌生人
- **解除 spam ban**：发邮件给 [recover@telegram.org](mailto:recover@telegram.org) 申诉，<strong>2-7 天回复</strong>，成功率约 50%
- **Premium 订阅扣费失败**：Apple ID 余额不足时不会重试，<strong>主动充值后再开</strong>
- **Premium 不可绑大陆 Apple Pay**：必须海外 Apple ID 余额或 USDT 支付
- **Telegram 安卓版被 Google Play 下架（部分国家）**：需 APK 直接装
- **2FA 密码忘了 = 永久失账号**：Telegram 不像 Google 可以"申诉找回"，2FA 密码<strong>必须备份</strong>
- **Telegram 服务器在欧盟 + 中东**：发文件 / 大群偶尔慢，<strong>挂欧洲节点最稳</strong>
- **Bot Token 不要泄漏**：BotFather 给的 Token = Bot 完全控制权，不要 commit 到 GitHub

## 六、账户安全

注册成功后立即：

1. Settings → Privacy and Security → **Two-Step Verification**（必开）
2. 设置 4-32 位密码 + 提示问题
3. **备份恢复邮箱**
4. **离线写下 2FA 密码**（极重要）
5. 关闭 "Last Seen"（隐私）
6. 只接受好友的消息（避免陌生人骚扰）
