## 一、注册前准备

### 1. 网络（核心）

| 节点 | 通过率 |
|------|------|
| 美国住宅 IP（海外 eSIM 流量）| **最高** |
| 日本 / 加拿大 / 澳大利亚住宅 IP | 高 |
| 美国数据中心 / 公共 VPN | 低 |
| **香港 / 台湾 / 新加坡** | 极低（Meta 区域风控严）|
| 大陆 IP | 0 |

> **海外 eSIM 数据流量** > 任何 VPN。giffgaff / 3HK / Clubsim 实体号开热点最稳。

### 2. 设备

- **iPhone 全新** > 已注册过老号的设备
- **没装过 IG App** 的全新手机最佳
- 安卓用户：清掉 Google Play Services 缓存 + 重置广告 ID
- ❌ 模拟器 / iCloud 多账号设备 / 越狱机 — 通过率极低

### 3. 海外手机号

- ✅ giffgaff / 3HK / Clubsim 自有 eSIM 号
- ✅ 借朋友海外手机号
- ⚠️ 接码平台 sms-activate 通过率约 40%
- ❌ +86 号几乎必拒

### 4. 邮箱

- ✅ 老 Gmail（美区）≥ 6 个月
- ✅ ProtonMail / Outlook
- ❌ 国内邮箱

### 5. 头像 + 简介

- 准备<strong>1 张真实人物 / 风景头像</strong>（**不要纯 logo / 二次元**，新号必拒）
- 准备<strong>2-3 行英文简介</strong>（不要中文，不要广告链接）
- 准备<strong>3-5 张本地相册照片</strong>（注册成功后立即发，证明是"真人"）

## 二、注册步骤

### 路径 A：iOS App（推荐）

1. iPhone 切到<strong>美区 Apple ID</strong>
2. App Store 下载 Instagram
3. **关 WiFi → 切到海外 eSIM 数据流量**
4. 打开 IG App → Sign Up
5. 手机号注册（国家选 US）→ 接 SMS
6. 设置用户名（**全英文 / 拼音，不要带数字结尾**）
7. 设置密码 ≥ 12 位
8. 出生日期 ≥ 18 岁
9. **立即上传头像**（关键！没头像的号被风控率高）
10. **立即写简介**
11. **关注 5-10 个公众账号**（任意头部账号即可）

### 路径 B：网页注册

1. Safari 打开 [instagram.com](https://www.instagram.com/)
2. 流程类似，但通过率比 App 略低

## 三、头 7 天养号策略（必须）

新号<strong>头 7 天</strong>不养，IG 算法会判定为机器人号直接封：

### Day 1（注册当天）
- 上传头像 + 简介
- 关注 5-10 个公众账号（明星 / 品牌 / 媒体）
- 浏览首页 Feed 5-10 分钟
- 给 3-5 个帖子点赞
- ❌ 不发帖、不评论、不私信

### Day 2-3
- 每天登录 1-2 次，每次 5-10 分钟
- 浏览 + 点赞 + 看 Reels
- 关注再 5 个账号
- ❌ 还是不发帖、不私信

### Day 4-5
- 发<strong>第一条 Story</strong>（生活照、风景照都行）
- 评论 1-2 条普通评论（不要广告关键词）
- 关注 5 个新账号

### Day 6-7
- 发<strong>第一条 Post</strong>（图片 + 简单英文文案）
- 给 5-10 个帖子点赞 + 评论
- 关注 5 个新账号
- 总关注数到 30+ 时算通过新手期

7 天后可以正常使用，但<strong>头 30 天</strong>仍要避免：
- 一次发 ≥ 5 条帖子
- 一天关注 ≥ 50 个账号
- 一次给 ≥ 20 条帖子点赞
- 一天发 ≥ 10 条私信

## 四、X 平台与 Reddit 高频踩坑

- **新号当天发广告 = 24h 封号**：IG 把"刚注册立即广告"标记为机器人，几乎必封
- **同 IP 注册多账号**：一天内同 IP 注册 2 个号 = 都被封；建议<strong>不同 IP / 不同时间</strong>
- **同手机号注册多账号**：一手机号最多 1-2 个 IG，超过被全家封
- **VPN 跳节点封号最多**：今天美国、明天日本 → IG 风控连封 + 强制 SMS 验证
- **批量关注 / 取关**：脚本化操作 = 立即封号
- **第三方提粉服务**：买粉买赞买浏览 = 永久封号
- **Reels 算法不友好新号**：新号 Reels 几乎没流量，建议<strong>头一个月只发 Story 和 Post</strong>
- **接码平台号 SMS 收不到**：IG 偶尔不发 SMS 给接码平台号
- **iCloud 备份恢复设备 = 关联识别**：原 iPhone 注过 IG，新 iPhone 用 iCloud 备份恢复后注册新号 = 立即被关联封
- **图片元数据泄漏 GPS**：发的照片 EXIF 含中国位置 = IG 推荐你给中国朋友 = 风控扣分
- **VPN 慢导致上传失败**：图片上传中断 = IG 标记"网络异常"
- **账号被封申诉**：90% 申诉无效；<strong>建议直接换设备 + 换号</strong>从头来过

## 五、账号安全

1. 注册完立即在 Settings → Security 开启 **2FA**（Authenticator App）
2. 设置 **备份邮箱**
3. 关闭 **Show activity status**（隐私）
4. **不要授权第三方 App 登录 IG**（除官方工具外）
5. 定期检查 **Login activity**

## 六、Threads / Reels / 其他 Meta 服务

注册成功 IG 后：
- **Threads**（Meta 的 X 替代品）— 用 IG 账号一键登录，不需要重新注册
- **Reels** 在 IG App 内，不是独立 App
- **Facebook** 与 IG 共享登录 — 但<strong>不要立刻关联</strong>，关联可能让 FB 风控波及 IG（详见 [Facebook 注册攻略](/docs/cross-border/overseas-accounts/facebook)）

- **大陆个人号转商业号**：Settings → Account → Switch to Professional account，<strong>需绑 Facebook 主页</strong>
