## 三关通关推荐顺序

新手按这个顺序最不易踩坑：

1. **先开第 1 关海外手机号**（giffgaff 最便宜稳定）— 因为后面所有账号都需要它收 SMS
2. **再开第 3 关美区 Apple ID + 美国 Gmail** — 它们是其他所有海外 App / 服务的根账号
3. **再开 IG / FB / YouTube / Telegram** — 注册时用到 Gmail 和海外手机号
4. **出门前再买第 2 关流量 eSIM** — 落地激活计时，别提前激活浪费天数

> **进阶**：如果你打算开海外银行 + 美股券商 + 加密交易所，参考 [出海资源直通车](/docs/guide) 一篇全套搞定。

> **网络层面**：在 VPS 上自建 3X-UI + Cloudflare WARP，让 ChatGPT / Claude 不掉线，详见 [VPS + 3X-UI 搭建](/docs/ai-era/ai-network/vps-3xui-setup)。
