# 普通人境外资产通关路线图

本路线图面向无港卡/无境外账户的中国大陆用户，梳理从**国内资金出境 → 境外账户落地 → 合规投资变现**的完整资金通道。所有路径均基于合规平台，红色/高成本路径已标注说明。

![资金流转概览](https://images.unsplash.com/photo-1559526324-593bc073d938?w=800&h=450&fit=crop)

## 一、无港卡用户资金路线图

以**长桥证券**为例，演示内地银行跨境汇入港/美股券商的完整路径（入金 + 出金）。

### 入金路径

```
国内主流银行卡（兴业 / 浦发 / 建设）
         │
         │ 跨境汇款，换汇约 100¥ 手续费
         ▼
iFAST Global Bank（英国 / 新加坡持牌银行）
  · 新加坡持牌银行，英镑 FPS 接收无费
  · 英镑无损，几乎秒到
         │
         │ 换币：英镑 → 新币，汇率最优
         ▼
Wise（多币种换汇核心）
  · 40+ 货币互转
  · 来源方为合规银行
         │
         ▼
美股券商终点
  · 盈透 IBKR（← SafePal / Bitget Wallet）
  · 嘉信 Schwab（← SafePal / Bitget Wallet）
  · 必贝 Webull
```

**不推荐路径**：长桥香港 ⇄ 长桥新加坡（券商互转无损 T+1）→ 长桥新加坡 → 新币入金（成本高，不推荐）

### 出金路径

```
美股券商
     │ 出金，新币无损
     ▼
Wise（出金中转，换币低损）
     │
     ▼
香港银行（中转节点：中银 / 汇丰 / 众安）
     │ 资金回国
     ▼
微信 / 支付宝（及其他国内银行卡）✅ 资金回国完成
```

## 二、交易所 — 合规券商 — 银行 — 加密卡 资金流转图

前头方向代表资金实际流向，五个核心环节依次串联。

### ① 交易所（加密资产起点）

| 交易所 | 特点 |
|--------|------|
| 欧易 OKX | 支持安全银行，流动性强 |
| 币安 Binance | 全球最大，支持安全银行 |
| Bybit | 主流合规交易所 |
| Bitget | 与 SafePal 钱包生态打通 |
| Gate.io | 小币种丰富 |
| 火币 HTX | 老牌平台 |
| Kraken | 合规交易所，支持安全银行 |
| Coinbase | 美国合规，仅受美国监管交易所 |

资金流出方式：USDT / USDC 出链 → 钱包

### ② 钱包 / 出入金工具

**热钱包（链上中转）**

| 钱包 | 推荐去向 |
|------|---------|
| SafePal Wallet | → 盈透 IBKR / 嘉信 Schwab |
| Bitget Wallet | → 盈透 IBKR / 嘉信 Schwab |
| OKX Wallet | 链上中转 |
| MetaMask | 链上中转 |

**法币 / 稳定币平台**

| 平台 | 功能 |
|------|------|
| BiyaPay | USDT → 法币，直接入金嘉信/IBKR，无需离岸账户 |
| StraitsX | ⇄ 新加坡银行卡，XSGD 稳定币法币互换 |
| BIT 交易所 | 法币兑换 |
| Kraken | 法币兑换 |

### ③ 法币通道（关键中转层）

**多货币法币通道**

| 货币 | 通道方式 |
|------|---------|
| 美元 USD | ACH → 华美银行 |
| 欧元 EUR | SEPA → N26 / Fiat24 |
| 英镑 GBP | FPS → iFAST / Paypal |
| 瑞郎 CHF | SIC → Fiat24 |

**iFAST Global Bank（英国持牌）**
- 仅接受美国合规交易所资金
- 英镑 FPS 接收，几乎秒到
- 适合从 Coinbase、Kraken 等合规交易所出金

**众安银行 ZA（香港）**
- 接受币安 / Kraken 直接入金
- 香港持牌银行，合规稳定

**卓锐证券（快捷路径）**
- 支持 USDT 直接存入港股账户
- 省去法币中转环节

Wise **不接受**交易所直接汇款，须先经过银行账户中转再使用 Wise 换汇。

### ④ Wise / 银行中转（换汇核心）

| 工具 | 特点 |
|------|------|
| **Wise** | 多币种换汇核心，40+ 货币互转，来源须为合规银行，汇率接近市场价，手续费约 0.3%~0.5% |
| **香港银行卡** | 中银 / 汇丰 / 众安，稳定中转，可出金回国 |
| **新加坡银行卡** | DBS / OCBC，⇄ StraitsX 法币互转 |
| **微信 / 支付宝** | 国内收款终点，资金回国完成 |

### ⑤ 券商终点（资产落地）

**美股券商**

| 券商 | 入金方式 |
|------|---------|
| 盈透 IBKR | ← SafePal / Bitget Wallet 直入 |
| 嘉信 Schwab | ← SafePal / Bitget Wallet 直入，可与 IBKR 互绑 |
| 必贝 Webull | 支持多种入金方式 |

**港股券商**

| 券商 | 备注 |
|------|------|
| 长桥新加坡 | 主流港美股平台 |
| 盈立新加坡 | 港股合规 |

## 四、完整通路总览

```
【国内资金】
国内银行卡
     │ 跨境汇款（换汇）
     ▼
【境外法币账户】
iFAST Global Bank / 众安银行 ZA / 香港银行
     │
     ├─ 英镑 FPS / 美元 ACH / 欧元 SEPA
     ▼
【换汇中转】
Wise（汇率最优，40+ 货币）
     │
     ├─────────────────────────────┐
     ▼                             ▼
【合规券商入金】              【加密通道入金】
IBKR / Schwab / Webull        交易所（OKX/Binance/Bybit 等）
 （美股 / 港股投资）               │
                                ├─ USDT → BiyaPay → 直入券商
                                ├─ USDT → 钱包 → SEPA → IBKR
                                └─ USDT → 加密卡 → 全球消费

【出金回国】
券商出金 → Wise 换汇 → 香港银行 → 微信/支付宝 ✅
```

## 五、路径选择建议

| 用户类型 | 推荐路径 |
|---------|---------|
| 首次出境资金，追求安全合规 | 国内银行 → iFAST → Wise → IBKR/Schwab |
| 持有 USDT，想直接入金券商 | USDT → BiyaPay → 嘉信 Schwab / IBKR |
| 需要境外日常消费卡 | Bitget Wallet × Fiat24（绑微信/支付宝） |
| 港股投资为主 | 众安银行 ZA → 长桥 / 盈立 / 熊猫证券 |
| 新加坡通道 | StraitsX ⇄ DBS/OCBC → 长桥新加坡 |
| 快速出金回国 | 券商 → Wise → 香港银行 → 微信/支付宝 |

**免责声明**：本路线图仅供信息参考，不构成投资或财务建议。各平台政策随时变化，请以官方最新规定为准。红色/高成本路径成本较高，非必要不推荐使用。
