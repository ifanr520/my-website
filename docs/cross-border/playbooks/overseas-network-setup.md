# 出海网络环境部署全攻略：三关通关指南

出海操作（交易所注册/开户/入金）的三大前提：**稳定的网络 → 海外 App 下载环境 → 境外手机号**，三关缺一不可，必须按顺序搭建。

## 第 2 关：海外应用环境——下载交易所/券商 App

核心问题：国内 App Store / Google Play 无法下载交易所、券商等海外应用，需切换海外账号或安装谷歌服务框架。

**iOS 方案：注册海外 Apple ID**

推荐注册美区或港区，步骤如下：

1. 打开 account.apple.com → 点击「创建 Apple ID」
2. 填写基本信息，**国家/地区选择「美国」或「香港」**
3. 绑定邮箱（未注册过的邮箱）+ 国内手机号接收验证码均可
4. 完成邮箱 + 手机号双重验证
5. 进入账号设置页 → Country/Region → 填写对应地区地址（美国可使用地址生成器生成免税州地址）

建议新注册一个专用海外 ID，不要将国区主力账号改区，避免影响国内 App 数据。

**下载 App 流程：**

登出 iPhone 上的国区 Apple ID → 登入海外 Apple ID → 进入 App Store 即可搜索并下载对应海外应用（OKX、Bybit、Binance、IBKR 等）。

**Android 方案：安装谷歌三件套**

国行安卓手机默认不含 Google 服务框架，需手动安装：

1. **谷歌服务框架（Google Services Framework）**
2. **谷歌账户管理程序（Google Account Manager）**
3. **Google Play 商店（Google Play Store）**

安装包可在酷安、机锋等安卓应用市场搜索「谷歌三件套」获取对应机型版本，安装后重启手机登录 Google 账号即可使用 Play Store。

## 第 3 关：海外 eSIM 号码——注册境外平台必备手机号

核心问题：注册交易所、券商、海外银行时，通常需要海外手机号接收验证码，国内号码无法通过验证。

**推荐英国 giffgaff，保号成本极低（每年仅 0.6 英镑），是性价比最高的长期持有方案。**

**方案一：giffgaff（强烈推荐，长期保号）**

giffgaff 是英国 O2 旗下虚拟运营商，**0 月租、无需实名**，每年仅需发一条短信（约 0.3 英镑）即可保号，是注册英美平台首选。

**申请激活步骤：**

1. 访问 giffgaff 官网申请免费 SIM 卡（填写转运地址或英国朋友地址）
2. 收到实体卡后，前往官网充值 **10 英镑**完成激活（支持 Visa/Mastercard）
3. 系统随机分配英国号码（+44 前缀），不满意可更改
4. 激活后即可接收短信验证码，收短信**完全免费**

**保号方法（每 6 个月操作一次）：**

- 发送一条短信（最便宜，每条 0.3 英镑）
- 或打一次电话 / 上一次网 / 充一次话费
