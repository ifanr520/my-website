# SafePal Card 申请全攻略：瑞士 IBAN 万事达卡从零到一

SafePal Card 是 SafePal 与瑞士金融科技公司 Fiat24 联合推出的万事达借记卡，背靠瑞士持牌银行独立 IBAN 账户，非企业预付卡，支持 USDT/USDC 充值、绑定微信/支付宝国内消费，也可直接入金盈透 IBKR / 嘉信 Schwab 等美股券商。

## 一、SafePal Card 核心优势

| 特性 | 说明 |
|------|------|
| **卡片类型** | 万事达（Mastercard）借记卡，瑞士 IBAN 独立账户 |
| **监管背景** | 瑞士 Fiat24 持牌运营，非企业预付卡，资金安全有保障 |
| **开卡 / 年费** | 全部**免费**，无最低余额要求 |
| **充值手续费** | **0**，且无上限（长期免费政策） |
| **666U 汇损补贴** | 每月前 666 USD 等值人民币消费汇损**全额补贴**，相当于 0 换汇损耗 |
| **绑定国内支付** | 支持绑定微信 / 支付宝，可用于淘宝、美团、滴滴等国内消费 |
| **直入美股券商** | 可通过 SafePal IBAN 账户直接 ACH/电汇入金盈透 IBKR / 嘉信 Schwab |
| **开卡激励** | 使用邀请码开户，可免费领取价值 69.9U 的 SafePal X1 硬件钱包 |

## 二、开卡前准备清单

- **身份证件（二选一）**：中国大陆身份证 或 护照（推荐护照，兼容性更好）
- **支持 NFC 的手机**：KYC 需用 NFC 扫描证件芯片
- **ReadID App**：护照/身份证 NFC 验证专用，提前下载
- **有效邮箱**：注册 Fiat24 账户接收验证码
- **科学上网工具**：全程需要能访问境外网络
- **海外区 Apple ID**：App Store 国区无法下载 SafePal App，需切换美区/港区账号
- **钱包内少量 ETH（Arbitrum 链）**：验证 Gas 费，约 $0.0001
- **至少 10 USD 等值资产**：激活万事达卡的最低要求

## 三、申请步骤详解

### Step 1：下载安装 SafePal App

前往 App Store（海外区账号）或 Google Play 下载 SafePal Wallet，安装完成后：

1. 点击「**创建新钱包**」
2. 设置 6 位数钱包密码（建议强密码）
3. **备份助记词**（12 个英文单词），离线抄写保存，切勿截图或云端存储

> 私钥钱包不支持申请 Bank 功能，只有助记词钱包才可使用。

### Step 2：进入 Bank 界面申请账户

打开 SafePal App → 顶部点击「**Bank**」→ 点击「**开始**」

> 如果顶部看不到 Bank 按钮，退出 App 重进，或切换科学上网节点后重试。

### Step 3：输入邀请码（关键，勿跳过）

点击「**我有邀请码**」，输入邀请码（如 **343000** 或 **782428**），可免费获赠价值 69.9U 的 SafePal X1 硬件冷钱包。

点击「下一步」→ 等待约 1 分钟完成账户创建 → 点击「**去 Fiat24 注册**」。

### Step 4：Fiat24 账户注册

系统跳转 Fiat24 注册流程：

1. 允许 App 获取**位置信息**（选「使用 App 时允许」）
2. 输入 SafePal 钱包密码 + 完成面容 ID / 指纹验证
3. 完成钱包**链上签名**授权（相当于电子签名绑定钱包地址）

### Step 5：KYC 身份认证（最重要环节）

打开页面内三个开关：**地址确认 / 身份证件 / GPS 定位**，点击继续。

**证件验证（护照为例）：**

1. 选择「Passport（护照）」或「Identity Card（身份证）」
2. 系统生成二维码 → **长按保存二维码图片**
3. 下载 **ReadID Ready App**（扫描二维码跳转或手动搜索下载）
4. 打开 ReadID App，输入护照信息（护照号、生日、有效期）
5. 将护照封面贴近手机背部，保持静止等待 NFC 读取成功（约 5~10 秒）
6. 完成活体**人脸识别**（光线充足，不戴口罩）
7. 返回 SafePal App，点击「**Click when you finish it**」

> **安卓用户**：NFC 读取前必须手动开启 GPS 定位，否则可能提示不支持；GPS 自动识别的地址不要修改，原样提交。

### Step 6：填写职业与开户信息

如实填写职业、收入来源信息，**开户用途建议全选**，提交后绑定邮箱，输入验证码完成最终签名确认。

审核通常 **几分钟内**通过，复杂情况需 1~2 个工作日。

### Step 7：申领万事达卡

银行账户审核通过后：

1. App 顶部点击「Bank」→「**银行卡**」→ 点击申领
2. 确保账户内**至少有 10 USD** 等值资产（任意币种）
3. 申领成功后可查看卡号、有效期、CVV
4. 开启「**Card Enabled**」完成激活

## 四、充值方式

充值支持多种稳定币，步骤：

1. 将交易所 USDT / USDC 提币到 SafePal 钱包（Arbitrum 链手续费最低）
2. 在 SafePal App Bank 界面点击「**充值**」
3. 选择 USDC 或 USDT，确认转入
4. 可选择在 App 内将稳定币兑换为美元 / 英镑 / 人民币等法币余额

> 充值手续费 0%，长期免费，无金额上限。

## 五、绑定微信 / 支付宝

与绑定普通银行卡流程完全一致：

1. 微信 / 支付宝 → 银行卡 → 添加银行卡
2. 输入 SafePal Card **卡号、有效期、CVV**
3. 输入手机短信验证码完成绑定
4. 绑定后即可用于国内日常消费，享受 666U 汇损补贴

## 六、用 SafePal Card 直接入金美股券商

SafePal Card 最强应用场景之一：**无港卡用户实现 USDT → 美股券商入金**。

**入金盈透 IBKR 流程：**

```
交易所 USDT
    │ 链上提币
    ▼
SafePal 钱包 → Bank → 稳定币兑换为 USD
    │ IBAN 电汇 / ACH
    ▼
盈透 IBKR / 嘉信 Schwab 账户
    ✅ 美股入金完成
```

> 进入 Bank → 点击中间转账按钮 → 选择「转账至外部账户」→ 填写 IBKR 的 IBAN 或 ABA 路由号 + 账号即可。

## 九、常见踩坑汇总

- **顶部没有 Bank 按钮**：退出重进 App，或切换梯子节点
- **NFC 识别失败**：取下手机壳，手机背面中上部贴紧证件，安卓须手动开 GPS
- **地址被改导致 KYC 失败**：GPS 自动识别地址原样提交，不要手动修改
- **重名风控黑名单**：遇风控无法注册，建议换身份（护照 vs 身份证）重新尝试
- **只显示 Visa 卡**：Visa 不支持中国消费，须先开通 Bank 账户再激活 Mastercard
- **单笔消费超 200 元扣 3%**：微信/支付宝对国际卡的平台规则，拆单消费可规避
- **充值后不能随意出金**：建议按需充值，避免资金长期停留在卡内
- **邀请码别漏填**：不填邀请码无法领取免费硬件钱包福利，且无法享受部分渠道专属优惠
