## X 平台高频踩坑

- **180 天冻结**：账户被风控（异常 IP / 退款率高 / 投诉多）会<strong>冻结资金 180 天</strong>，期间无法提现。预防方法：注册稳定 IP、不频繁切换地区、避免大额突发收款
- **退款率 > 1% 触发审查**：电商账户尤其注意，退款率超标会触发限制 + 资金保留 90 天
- **跨境提现汇率差**：PayPal 内部汇率比市场价差 <strong>2~3%</strong>，<strong>建议提现到 Wise USD 账户</strong>而非直接换 CNY 提到国内银行
- **首次大额收款被冻结**：新账户突然收 > $1000 易触发审查，建议<strong>从 $50~$200 小额开始建立信用</strong>
- **关联账户被全部封**：同一身份证 / 银行卡 / IP 注册多个账户，<strong>所有关联账户全部冻结</strong>
- **HTTPS 钓鱼**：每月都有"PayPal 客服"假邮件钓鱼。<strong>所有官方通讯只走 paypal.com 站内信箱</strong>
