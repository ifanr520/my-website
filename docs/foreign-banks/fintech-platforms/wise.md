## X 平台高频踩坑

- **加密来源资金被退回**：Wise<strong>不接受</strong>来源为交易所的转账（OKX / Binance / Bybit 等），<strong>必须先入银行账户再汇 Wise</strong>。违反会被冻结审查
- **大额需提供资金来源**：单笔 > £10,000 / $10,000 等值，Wise 会要求合同 / 工资单 / 报税单等证明
- **Wise 卡 ATM 取现限额**：每月免费 £200 ATM 取现，超出 1.75% + £0.50/笔
- **Wise 不是银行**：账户余额不享受 FSCS 存款保险，Wise 是<strong>电子货币机构（EMI）</strong>，资金隔离托管在 JP Morgan / Barclays 等
- **国内手机号验证慢**：建议用 Gmail + 国内手机号 / 海外手机号均可，<strong>不要用 QQ 邮箱</strong>
- **多账户被关**：一个身份只能开一个 Personal + 一个 Business，多账户会被全部关闭
