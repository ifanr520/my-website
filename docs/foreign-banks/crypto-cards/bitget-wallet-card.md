# Bitget Wallet Card 申请全攻略：瑞士 IBAN 万事达卡从零到一

Bitget Wallet Card 是 Bitget Wallet（原 BitKeep）与瑞士持牌金融科技公司 **Fiat24** 联合推出的万事达借记卡，拥有独立的瑞士 IBAN 账户，0 开卡费、0 年费，支持身份证注册，绑定微信/支付宝/Apple Pay 全覆盖。**注意：Bitget Wallet Card 与 Bitget 交易所 Bitget Card 是两个完全不同的产品，本篇介绍的是钱包版（Fiat24 版）。**

## 一、核心优势速览

| 特性 | 说明 |
|------|------|
| **卡片类型** | 万事达（Mastercard）借记卡，瑞士 Fiat24 独立 IBAN 账户 |
| **发卡机构** | 瑞士 Fiat24 持牌运营，受瑞士金融监管局（FINMA）监管 |
| **开卡 / 年费** | 全部**免费**（使用定制邀请码） |
| **充值手续费** | **0%**（USDC/USDT 充值免费） |
| **月 0 Fee 额度** | 默认 **$400/月**；使用邀请码可升至 $600 甚至 $800 |
| **国内绑定支付** | 支持绑定微信支付、支付宝、Apple Pay |
| **人民币消费** | 单笔 ≤200 元**完全免手续费**；>200 元支付宝/微信收取 3%（平台规则） |
| **非美元消费费率** | 月免费额度内 **0%**；超额后 **1.7% FX Fee** |
| **KYC 要求** | 身份证可直接认证，**无需护照**（目前中国大陆可用） |
| **瑞士 IBAN** | 独立账号，可收跨境电汇 |

## 三、开卡前准备清单

- **中国身份证**：正反面清晰照片（无需护照）
- **有效邮箱**：Gmail / Outlook 均可，用于 Fiat24 账户注册
- **支持 NFC 的手机**：KYC 阶段需 NFC 读取身份证芯片
- **ReadID Ready App**：身份证 NFC 验证专用，提前下载
- **科学上网环境**：全程需境外 IP，建议挂台湾/香港节点
- **至少 10 USDC**：激活虚拟卡的最低余额要求（资金不消失，仍属于你）
- **少量 USDC 多余 1 枚**：用于支付激活时 Gas 费，建议充值 **11 USDC**
- **Arbitrum 链少量 ETH**：约 $0.001，用于链上授权操作的 Gas 费

## 四、申请步骤详解

### Step 1：下载安装 Bitget Wallet App

前往 App Store（需海外区 Apple ID）或 Google Play 搜索「**Bitget Wallet**」，安装最新版本。

1. 打开 App → 点击「创建新钱包」
2. 设置 6 位钱包密码
3. **备份助记词**（12 个英文单词），务必离线抄写，切勿截图或存入云盘

### Step 2：进入银行卡申请入口

打开 Bitget Wallet App → 底部导航「**钱包**」→「**银行卡**」→「**立即申请**」

> 如果找不到「银行卡」入口，尝试退出重进 App，或切换 VPN 节点后刷新。

### Step 3：注册 Fiat24 账户

1. 点击「申请借记卡」→ 输入邀请码（如有，可解锁 $600 免费额度及开卡奖励）
2. 填写姓名（与身份证一致）、出生日期、邮箱
3. 点击「Submit」→「去认证」

### Step 4：KYC 身份证认证（核心环节）

1. 证件类型选「**Identity Card（身份证）**」
2. 页面生成二维码 → **长按保存二维码图片到相册**，同时记下 Request ID
3. 下载 **ReadID Ready App**（扫码跳转或手动搜索）
4. 打开 ReadID，输入身份证号、出生日期、有效期
5. 将身份证贴近手机背面中上方，静止等待 NFC 读取（约 5~10 秒）
6. 完成活体**人脸识别**（光线充足，正面朝向）
7. 返回 Bitget Wallet App，点击「**Click when you finish it**」

**⚠️ 安卓用户注意：**
- NFC 读取前必须手动开启 **GPS 定位**，否则可能报错
- GPS 自动识别地址**不要手动修改**，原样提交，否则 KYC 大概率失败

### Step 5：充值激活虚拟卡

1. 将 **USDC（Arbitrum 链）** 充值到 Bitget Wallet（至少 **11 USDC**）
2. 在银行卡页面点击「**充值**」→ 选择 USDC → 输入金额 → 确认
3. 充值到账后点击「**去激活**」→「Activate Card」→ 开启「Card Enabled」
4. 激活完成后即可查看完整卡号、有效期、CVV

> ⚠️ 充值**必须选 Arbitrum 链**，选错其他链资产将无法到账且难以找回。

### Step 6：绑定微信 / 支付宝 / Apple Pay

**支付宝 / 微信（流程一致）：**
1. 支付宝 / 微信 → 银行卡 → 添加银行卡
2. 输入 Bitget Wallet Card 的**卡号、有效期、CVV**
3. 输入短信验证码完成绑定

**Apple Pay：**
1. iPhone「钱包」App → 右上角「+」→ 添加借记卡
2. 输入卡号信息绑定，可在 Apple Pay 支持的任意场景消费

## 七、常见踩坑汇总

- **链选错导致资产丢失**：充值必须选 **Arbitrum 链 USDC**，其他链（如 Tron/BSC）充值的资产无法到账，几乎无法找回
- **月免费额度不检查就超额**：每月 1 号在 App 查看「额度使用进度」，快到上限时切换国内银行卡消费，省着额度用
- **单笔 >200 元触发 3% 手续费**：小额日常消费影响不大，但大额消费（买电子产品、充话费）白白亏 3%，建议分拆或换其他支付方式
- **微信绑卡被风控**：部分用户反映直接绑微信会出现风控拒绝，可改绑**美团、抖音、拼多多**等 App，消费效果相同，风控概率低很多
- **GPS 地址被修改导致 KYC 失败**：ReadID 自动识别的地址原样提交，不要改动任何字符
- **NFC 读取失败**：取下手机保护壳，确保身份证正面贴近手机背面靠近摄像头区域，安卓必须开 GPS
- **微信年度限额 15,000 元用完**：微信对国际卡年度限额约 15,000 元，用完当年无法通过微信消费，需等次年 1 月重置；支付宝限额 7 万元相对宽裕
- **充值资产目前无法转出**：卡内余额只能用于消费，无法提回钱包，**务必按需充值，切勿大额存入**
- **Bitget Card vs Bitget Wallet Card 混淆**：交易所 Bitget Card 是 Visa 卡，需 VIP 资质；钱包版 Bitget Wallet Card 是 Mastercard，0 门槛，两者完全不同，申请入口也不同
