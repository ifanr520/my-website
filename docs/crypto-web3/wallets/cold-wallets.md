## 通用避坑（X 平台高频踩坑）

- **二手设备一律不买**。任何来源的二手硬件钱包都可能被预设助记词陷阱
- **助记词必须离线手抄 / 钢板备份**，<strong>截图 / 云盘 / 微信收藏 = 公开</strong>
- **官方原箱 + 防伪贴**，开箱前先核对官网防伪查询
- **首次设置一定生成新助记词**，不要导入旧助记词到新设备
- **PIN 码 ≥ 6 位**，连续输错触发自毁是好事
- **不要在咖啡馆等公共 Wi-Fi 操作**链接钱包初始化
- **品牌官网下单**：OneKey 直邮、Ledger / Trezor / BitBox 走转运（选择重视隐私的转运公司）
- **Tangem 一套 3 张分别保管**，别全部放一个抽屉
