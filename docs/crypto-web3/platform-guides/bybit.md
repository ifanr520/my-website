## 六、X 平台高频踩坑警告

- **跟单交易盲目复制顶级交易员**：榜单顶部多为短期高收益账号，回撤期跟单等于接盘。建议看交易员 90 天累计收益 + 最大回撤，不看 7 天榜
- **合约自动平仓机制**：Bybit 维持保证金率比 Binance 略低，看似杠杆更高，但**爆仓滑点也更大**
- **微信支付被封**：Bybit Card 微信绑定 2024 年已被封，不要再尝试，浪费冻卡额度
- **"高 APR 资金费率套利"陷阱**：X 上有人推 100% APR 套利，多数是新币短期资金费率，结算前可能反向，**新手不要碰资金费率套利**
- **API Key 泄漏**：连第三方机器人时 API 仅给「读 + 交易」，**绝不勾选「提币」**
- **平台稳定性**：行情剧烈波动时偶有"卡顿无法平仓"案例，重要持仓建议设硬止损 + 不开满仓

## 七、出金路径

- **加密侧**：USDT (TRC20) → 钱包 → SafePal / Bitget Wallet 卡 → 直接消费
- **法币侧**：P2P 卖出 → 国内银行卡（建议三大行 + 中小行轮换收款，避免单卡触发风控）
- **跨平台**：USDT (Arbitrum) → IBKR / 嘉信美股账户（通过 SafePal 中转）
