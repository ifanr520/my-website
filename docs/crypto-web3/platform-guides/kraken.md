## 五、入金路径

### 推荐路径：USD ACH（美国用户）

- 绑定美国银行账户
- ACH 转账，1~3 工作日到账，**手续费 0**
- 适合 Schwab / Chase / Bank of America 等大行账户

### 欧元 SEPA / 英镑 FPS

- SEPA：欧盟用户首选，**T+0 到账，手续费几乎为零**
- FPS：英国用户首选，**iFast Global Bank、Revolut、Wise** 都可中转

### USDT 链上充值

- 「Funding」→「Deposit」→ USDT
- 链选 ERC20 / TRC20 / Solana
- ERC20 手续费高（5~20 USD），**TRC20 最便宜**

### 加密资产入金（推荐组合）

```
其他交易所 USDT (TRC20) → Kraken USDT
        ↓
Kraken 兑换为 USD / EUR / GBP
        ↓
ACH / SEPA / FPS 出金到 iFast / Wise / 银行
```

## 六、X 平台高频踩坑警告

- **大陆 IP 注册被拒**：必须海外网络环境，**且海外居住地址证明真实可查**，否则即使 KYC 提交也会被拒
- **客服全英文**：Kraken 客服仅英文，**复杂问题需邮件沟通**，平均响应 24~48h
- **资金来源审查严格**：大额入金需提供 source of funds，**P2P 来源资金可能被冻结审查**
- **法币入金银行匹配**：必须使用**与 KYC 同名**银行账户入金，第三方银行入金会被退回甚至冻结
- **Pro 平台与 Lite 平台数据分离**：新用户首次只能用 Lite，**手续费比 Pro 高一倍**，记得切换到 Pro 平台交易
- **美国用户期货 / 合约限制**：Kraken Futures 美国本土用户不开放，**仅美国境外可用**
- **客户端登录失败**：Kraken App 偶尔需要 VPN 切换节点才能正常登录
