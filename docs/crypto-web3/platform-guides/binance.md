## 二、注册前准备

- **身份证件**：护照（推荐）或中国大陆身份证。X 平台反馈：用护照 KYC 通过率最高，身份证目前对部分大陆用户限制更多。
- **境外网络环境**：境内 IP 已被拉黑，必须挂海外节点（推荐香港、新加坡、日本三选一，避免欧美 IP 触发额外审查）。
- **干净邮箱**：Gmail / ProtonMail，**不要用国内手机号绑定的 QQ / 163**。
- **海外手机号或可接收 SMS 的虚拟号**：Google Voice、Tello、AT&T eSIM 均可。
- **Google Authenticator**：双因子验证强制要求。

## 四、入金路径

### 路径 A：USDT 链上充值（推荐）

1. 从交易所或钱包提 USDT 到 Binance 充币地址
2. **链选 BSC（BEP20）或 Tron（TRC20）**，BSC 手续费几乎为零，TRC20 约 1 USDT
3. 切勿选 ERC20，主网 Gas 费可能高达 5~20 USD

### 路径 B：法币 P2P 交易

- 进入「P2P」→ 选 USDT/CNY → 选「微信 / 支付宝 / 银行卡」付款
- 优先选**已认证商家**（蓝色盾牌 + 完成 1000 单以上）
- 单笔不超过 5 万元 RMB，避免触发国内银行风控

### 路径 C：信用卡 / 借记卡入金

- 支持 Visa / Mastercard，但**国内卡基本被拒**
- 海外卡（如 SafePal Card / Bitget Wallet Card）可成功，手续费约 1.8%

## 六、X 平台高频踩坑警告

- **同一身份多账号**：被检测到一个身份证 / 护照绑定多个账户，**所有账户冻结资产清零**，申诉难度极大
- **大额 P2P 收款**：单日收款 >5 万 RMB 极易被国内银行风控冻卡，建议分次、跨银行收
- **合约爆仓后情绪化加仓**：X 上每月都有"100 倍杠杆全亏"的悲剧，**永远不要用全部资金做合约**
- **公开晒 BNB / 持仓**：X 平台已有钓鱼团队按公开持仓信息精准 social engineering，建议小号晒、大号沉默
- **用国内 VPN 注册再切回境外**：风控系统记录设备指纹和 IP 历史，注册环境**必须自始至终是同一海外节点**
- **客服钓鱼私信**：Binance 官方**永远不会主动私信**，所有客服只在 App 内对话，X 私信、Telegram 群里说"我是 Binance 客服"的 100% 是骗子
