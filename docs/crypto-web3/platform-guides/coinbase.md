## 二、Coinbase 是否适合你

**适合**：
- 持美国 SSN / 美国银行账户
- 需要把交易所账户和券商账户（Schwab、Fidelity）合规衔接
- 重视审计透明度，机构 / 退休账户合规要求严
- 美国本土用户希望 1099 表格简化报税

**不适合**：
- 中国大陆纯境内用户（**完全无法注册**）
- 需要合约 + 高杠杆（Coinbase 美国本土无合约）
- 需要小币种和打新（Coinbase 上线币种保守）

## 六、入金路径

### 推荐路径：USD ACH（美国用户）

- 绑定美国银行账户
- ACH 转账，1~3 工作日到账，**手续费 0**
- 单日入金上限 ~ $25,000，单月 ~ $50,000

### Wire 电汇

- 适合大额入金（> $25,000）
- 手续费 $10/笔，1 工作日到账

### USDT / USDC 链上充值

- USDC 在 Coinbase 是原生稳定币，**入出金免费**
- USDT 走 ERC20 / Solana / Polygon

### Coinbase Card（部分用户）

- 美国 + 部分欧洲国家可申
- Visa 借记卡，加密资产消费
- **最高 4% BTC 返现** 或 ETH 返现

- **中国大陆 IP 100% 拒绝**：Coinbase 风控系统已识别 99% 大陆 VPN 出口 IP，**用国内 IP 一旦被检测，账户立即冻结**

- **税务报告自动同步 IRS**：美国用户所有收益**自动报送 IRS**，**任何避税尝试都会被追溯**
- **Coinbase Wallet 与交易所分离**：Coinbase Wallet 是独立 App，**助记词由你自己保管**，丢了 Coinbase 也无法找回
- **NFT 板块停摆**：Coinbase NFT 已停止运营，**不要在 Coinbase 内操作 NFT**
- **Vault 提币 48h 锁定**：开启 Vault 后提币有 48h 锁定期，**急用资金的不开**
