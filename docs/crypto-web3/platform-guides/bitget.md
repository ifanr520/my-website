## 四、跟单交易上手（Bitget 招牌玩法）

1. App 底部「**跟单**」
2. 排行榜选交易员，看以下三个核心指标：
   - **90 天累计收益率**（短期高收益常是 Pump，避开）
   - **最大回撤**（>40% 直接 pass）
   - **跟随人数稳定增长**（不是暴涨暴跌）
3. 点「跟随」→ 设置跟单金额（建议从 50~100 USDT 起步）
4. 设定**单笔最大亏损上限**（比如不超过 10 USDT），防止交易员剧烈操作

> X 平台高频教训：**所谓"千倍交易员"绝大多数是用账号刷量制造排名**，跟随后高概率横盘 / 大幅回撤。建议**至少观察 30 天**再小额跟单。

### 链上 USDT（推荐）

- 「资产」→「充币」→ USDT
- 链选 **TRC20 / BSC / Arbitrum**

- 中国大陆部分时段 P2P 区被屏蔽，**推荐用 USDT 链上转入而非 P2P**

## 七、X 平台高频踩坑警告

- **跟单"老师"群拉人**：X 上有大量"Bitget 老师群"鼓动跟单 + 加杠杆，**100% 是诱导爆仓后割粉**
- **小币种现货流动性陷阱**：Bitget 上线新币速度快，但**部分小币种买入容易卖出难**，深度极差
- **跟单合约自动倍率**：跟单时若交易员开 50 倍杠杆，你的本金风险被放大同样倍数，**建议固定低倍率跟单**

- **空投 Launchpool 休眠**：参加打新后忘记领取，**奖励 90 天后失效自动归还项目方**
- **客服私信**：Bitget 官方**永远不在 Telegram / X 主动私信**，所有客服在 App 内对话
